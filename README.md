# 字幕遮挡器应用项目说明

## 主要功能
- 安卓端遮挡视频字幕，提升观影体验。
- 支持按住框体移动位置
- 白色角点拖入扩大或缩小范围
- 白色右边线拖入调宽度
- 白色下边线拖入调长度

## 项目结构
- `app/`：主应用模块，包含源码、资源和构建配置。
- `build.gradle`、`settings.gradle`：项目级构建配置文件。
- `gradle/`：Gradle相关配置。

## 构建与运行
1. 安装 Android Studio 或配置好 Android 开发环境。
2. 使用 `./gradlew assembleRelease` 命令编译生成 Release APK。
3. APK 文件生成于 `app/build/outputs/apk/release/` 目录。
